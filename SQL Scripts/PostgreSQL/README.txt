Скрипты для последовательного наката (PostgreSQL).
В корневой папке лежит только скрипт для создании самой базы данных.
Все последующие скрипты лежат в соответствующих папках.

----------------------------------------------------------------------------------
Описание скриптов
----------------------------------------------------------------------------------
0001_CREATE_DATABASE.sql - Создание рабочей базы данных.
0002_CREATE_TABLE_Legends - Создание таблицы с основной сущностью - Легенда
